# Keygen Portal

## Developing

### Environment

```sh
VITE_KEYGEN_ACCOUNT_ID=1d6c6f07-6caf-4274-933c-9a360f0d03a2
VITE_KEYGEN_MODE=singleplayer
VITE_KEYGEN_HOST=api.keygen.localhost
VITE_KEYGEN_VERSION=1.8
```

### Setup

To install dependencies, run:

```bash
pnpm install
```

### Running

To start the app, run:

```bash
pnpm run dev
```

### Testing

To run the test suite, run:

```bash
pnpm run test
```
